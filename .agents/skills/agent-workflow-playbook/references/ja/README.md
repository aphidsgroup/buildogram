# AI エージェント ワークフロー プレイブック — スキルアーキテクチャ & プラグインマーケティング

AI エージェント自動化システム設計の完全ガイド。スキル/プラグインアーキテクチャ、マルチエージェント連携パターン、マーケットプレイス配信戦略、ワークフローオーケストレーションのベストプラクティスを網羅。

## 学べること
- スキルアーキテクチャパターン（単一目的 vs. オーケストレーターエージェント）
- プラグインマーケットプレイスの SEO と配信
- マルチエージェントワークフロー設計（逐次・並列・階層型）
- エージェント間通信プロトコル
- AI エージェントスキルのマネタイズ戦略

## 対象読者
- エージェントシステムを構築する AI 開発者
- マーケットプレイスにスキル/プラグインを公開するチーム
- エージェントネイティブプロダクトを設計するプロダクトマネージャー

## 主要コンセプト

### スキルアーキテクチャパターン
| パターン | ユースケース | 複雑度 |
|----------|-------------|--------|
| 単一目的 | 1タスク、1スキル | 低 |
| オーケストレーター | 複数サブスキルの連携 | 中 |
| 階層型 | 多階層エージェント委任 | 高 |
| リアクティブ | イベント駆動スキルトリガー | 中 |

### マルチエージェントワークフロー設計
- **逐次型**: タスク A → タスク B → タスク C（パイプライン）
- **並列型**: タスク A・B・C を同時実行し結果をマージ
- **階層型**: マネージャーエージェントがスペシャリストに委任
- **イベント駆動型**: 外部トリガーに反応してスキルが起動

### マーケットプレイス配信
- スキル説明文を検索向けに最適化
- 明確な命名規則を使用
- README に使用例を含める
- 関連キーワードでタグ付け
