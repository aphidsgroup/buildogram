# KOL 콜드 아웃리치 템플릿 — 답장받는 DM 스크립트

더 이상 무시당하지 마세요. 이 템플릿은 500건 이상의 성공적인 크리에이터 파트너십으로 검증되었습니다.

## DM 스크립트

플랫폼별 템플릿 (Twitter, Instagram, LinkedIn):

- **Twitter DM 공식**: 훅 (구체적인 콘텐츠 칭찬) → 브릿지 (공유 맥락) → 요청 (명확하고 부담 없는)
- **Instagram DM**: 먼저 최근 게시물 3개에 댓글 → 48시간 대기 → 비주얼 에셋 첨부해서 DM
- **LinkedIn InMail**: 프로페셔널한 톤, 상호 커넥션 언급, 명확한 가치 제안
- **핵심 원칙**: 첫 문장을 개인화, 80단어 이내, CTA는 하나만
- **타이밍**: 화~목요일, 상대 시간대 기준 오전 9~11시에 전송

## 이메일 시퀀스

답장률 40% 이상의 3단계 콜드 이메일 템플릿:

- **이메일 1 (Day 0)**: 제목: "[상대의 최근 콘텐츠] + 아이디어" — 개인화된 칭찬 + 가치 제안 + 소프트한 요청
- **이메일 2 (Day 3)**: "리마인드 — [새로운 앵글 또는 소셜 프루프 추가]" — 간결함이 승리
- **이메일 3 (Day 7)**: "마지막 메시지 — [긴급성 또는 독점성 제시]" — 명확한 데드라인 포함 최종 시도
- **제목 공식**: 상대 이름 사용, 콘텐츠 언급, 스팸 트리거 회피
- **기술 설정**: 커스텀 도메인, SPF/DKIM, 대량 발송 전 워밍업 기간

## 가치 제공 우선 접근법

크리에이터가 답장하고 싶게 만드는 피칭 방법:

- **상대가 얻는 것부터 시작**: 독점 접근, 오디언스에 대한 가치, 크리에이티브 자유, 수익
- **리서치했음을 보여주세요**: 구체적인 영상, 게시물, 오디언스 데모그래픽 언급
- **YES를 쉽게 만드세요**: 미리 만든 에셋, 유연한 타임라인, 낮은 노력 요구
- **소셜 프루프**: 다른 파트너 크리에이터 이름을 언급 (허락받은 경우)
- **포맷 옵션**: 2~3가지 콜라보 형식을 제시하고 상대에게 선택하게 하세요

## 팔로업 리듬

언제 팔로업하고, 몇 번 보내고, 언제 멈출지:

- **3터치 룰**: 하나의 채널에서 3회 이상 팔로업 금지
- **간격**: Day 0, Day 3, Day 7 — 거기서 멈춤
- **채널 전환**: 이메일 3회 무응답이면 DM 1회 시도 (동시 진행 금지)
- **시즌 인식**: 공휴일, 제품 론칭 주간, 바쁜 시기는 피하세요
- **품격 있는 퇴장**: 마지막 메시지는 압박 없이 문을 열어두세요

## 파트너십 구조

유료 vs 물물교환 vs 어필리에이트 — 언제 무엇을 제안할지:

- **고정 보수**: 대형 크리에이터 (10만+ 팔로워)에 최적; 예측 가능한 비용; CPM $15~50으로 협상
- **제품 물물교환**: 초기 단계에 최적; 프로 액세스를 무료 제공하고 솔직한 리뷰와 교환
- **어필리에이트/수익 공유**: 장기 관계에 최적; 15~30% 리커링 커미션
- **하이브리드 (고정 + 어필리에이트)**: 초기 리스크 감소; 지속적 프로모션 인센티브 정렬
- **성과 기반**: 가입 또는 전환당 지불; 트래킹 인프라가 있을 때 최적
