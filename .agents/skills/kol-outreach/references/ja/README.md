# KOLコールドアウトリーチテンプレート — 返信が来るDMスクリプト

もう無視されるのは終わりです。これらのテンプレートは500件以上のクリエイターパートナーシップで実証済みです。

## DMスクリプト

プラットフォーム別テンプレート（Twitter、Instagram、LinkedIn）：

- **Twitter DMの公式**: フック（具体的なコンテンツを褒める）→ ブリッジ（共有コンテキスト）→ アスク（明確で低コミットメント）
- **Instagram DM**: まず最近の投稿3つにコメント → 48時間待つ → ビジュアルアセットを添付してDM
- **LinkedIn InMail**: プロフェッショナルなトーン、共通コネクションへの言及、明確なバリュープロポジション
- **基本原則**: 最初の一文をパーソナライズ、80語以内、CTAは1つだけ
- **タイミング**: 火〜木曜日、相手のタイムゾーンで午前9〜11時に送信

## メールシーケンス

返信率40%以上の3ステップコールドメールテンプレート：

- **メール1（Day 0）**: 件名: 「[相手の最近のコンテンツ] + アイデア」 — パーソナライズした褒め + 価値提案 + ソフトなお願い
- **メール2（Day 3）**: 「リマインドです — [新しい角度かソーシャルプルーフを追加]」 — 簡潔さが勝つ
- **メール3（Day 7）**: 「最後のご連絡 — [緊急性か限定感を作る]」 — 明確な期限付きの最終アプローチ
- **件名の公式**: 相手の名前を使う、コンテンツに言及、スパムトリガーを避ける
- **技術セットアップ**: カスタムドメイン、SPF/DKIM、大量送信前のウォームアップ期間

## バリューファーストアプローチ

クリエイターが返信したくなるピッチの仕方：

- **相手が得るものから始める**: 独占アクセス、オーディエンスへの価値、クリエイティブの自由、収益
- **リサーチ済みであることを示す**: 具体的な動画、投稿、オーディエンス属性に言及
- **YESを言いやすくする**: 事前に作ったアセット、柔軟なタイムライン、低い労力
- **ソーシャルプルーフ**: 他にパートナーした（許可済みの）クリエイター名を挙げる
- **フォーマットの選択肢**: 2〜3つのコラボ形式を提示し、相手に選ばせる

## フォローアップのリズム

いつフォローアップし、何回送り、いつ止めるか：

- **3タッチルール**: 1つのチャネルで3回以上のフォローアップは送らない
- **間隔**: Day 0、Day 3、Day 7 — そこで止める
- **チャネル切り替え**: メールで3回反応なしなら、DMを1回試す（同時進行はしない）
- **季節の意識**: 祝日、プロダクトローンチ週、繁忙期は避ける
- **潔い退出**: 最後のメッセージはプレッシャーなく扉を開けたままにする

## パートナーシップ構造

有料 vs バーター vs アフィリエイト — いつ何を提案するか：

- **固定報酬**: 大型クリエイター（10万+フォロワー）に最適; コスト予測可能; CPM $15〜50で交渉
- **プロダクトバーター**: アーリーステージに最適; プロアクセスを無料提供し正直なレビューと交換
- **アフィリエイト/レベニューシェア**: 長期関係に最適; 15〜30%のリカーリングコミッション
- **ハイブリッド（固定+アフィリエイト）**: 初期リスク低減; 継続プロモーションのインセンティブが一致
- **成果報酬型**: サインアップやコンバージョンごとに支払い; トラッキング基盤がある場合に最適
