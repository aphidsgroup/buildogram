# B2B 마케팅 플레이북 — LinkedIn + 콜드 이메일 + 웨비나 파이프라인

B2B SaaS를 위한 3가지 검증된 채널을 하나의 반복 가능한 파이프라인으로 통합.

## LinkedIn 엔진

일관된 LinkedIn 활동으로 권위를 구축하고 인바운드 리드를 획득합니다.

- **콘텐츠 리듬**: 주 3~5회 게시 (소트 리더십, 전술적 하우투, 사회적 증거를 조합)
- **인게이지먼트 포드**: 10~20명의 동료와 초기 인게이지먼트 시그널 조율
- **DM→데모 플로우**: 3회 이상의 터치포인트 후 따뜻한 접근, 시간을 요청하기 전에 가치 제공
- **프로필 최적화**: 헤드라인 = 제공하는 성과, 배너 = 사회적 증거, 피처드 = 리드 마그넷

## 콜드 이메일 시퀀스

개인화된 멀티터치 시퀀스를 통한 예측 가능한 아웃바운드 파이프라인.

- **제목**: 3~5단어, 소문자, 호기심 또는 관련성 훅 (오픈율 35% 이상 목표)
- **개인화 레이어**: 기업 트리거 이벤트, 역할별 페인 포인트, 상호 인맥
- **후속 타이밍**: 1일차→3일차→7일차→14일차 (최소 4회 터치)
- **CTA 최적화**: 처음엔 소프트한 요청("대화할 가치가 있을까요?"), 후속에서 하드한 요청

## 웨비나 퍼널

교육 콘텐츠를 통한 높은 구매 의도의 리드 생성.

- **주제 선정**: ICP가 이번 분기에 직면하는 구체적인 문제를 해결
- **프로모션 믹스**: 이메일 리스트(40%), LinkedIn 게시물(30%), 파트너 크로스 프로모션(30%)
- **웨비나 후 너처링**: 리플레이 + 핵심 정리 이메일→케이스 스터디→데모 오퍼 시퀀스
- **전환 벤치마크**: 등록→참석 30~40%, 참석→SQL 10~15%

## ABM 레이어

엔터프라이즈 딜(ACV $50K 이상)을 위한 어카운트 기반 타겟팅.

- **타겟 어카운트 리스트**: 최대 50~100개 어카운트, 분기별 갱신
- **멀티스레딩**: 어카운트당 3~5명의 이해관계자에게 동시 접근
- **커스텀 콘텐츠**: 개인화된 랜딩 페이지, ROI 계산기, 산업별 케이스 스터디
- **시그널 트래킹**: 인텐트 데이터, 웹사이트 방문, 콘텐츠 인게이지먼트 스코어링

## 메트릭스 대시보드

선행 지표와 후행 지표로 파이프라인 건강도를 추적합니다.

- **파이프라인 벨로시티**: 딜이 단계를 통과하는 속도
- **SQL 비율**: 마케팅 적격→세일즈 적격 전환율
- **CAC 벤치마크**: 채널별, 세그먼트별, 시계열 트렌드
- **매출 어트리뷰션**: 정확한 채널 ROI를 위한 멀티터치 어트리뷰션 모델
