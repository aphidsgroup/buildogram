# B2Bマーケティングプレイブック — LinkedIn + コールドメール + ウェビナーパイプライン

B2B SaaS向けの3つの実証済みチャネルを1つの再現可能なパイプラインに統合。

## LinkedInエンジン

一貫したLinkedInプレゼンスで権威を構築し、インバウンドリードを獲得する。

- **コンテンツリズム**: 週3〜5回投稿（ソートリーダーシップ、戦術的ハウツー、社会的証明を組み合わせる）
- **エンゲージメントポッド**: 10〜20人のピアと初期エンゲージメントシグナルを調整
- **DM→デモフロー**: 3回以上のタッチポイント後にウォームアプローチ、時間を要求する前に価値を提供
- **プロフィール最適化**: ヘッドライン＝提供する成果、バナー＝社会的証明、フィーチャード＝リードマグネット

## コールドメールシーケンス

パーソナライズされたマルチタッチシーケンスによる予測可能なアウトバウンドパイプライン。

- **件名**: 3〜5語、小文字、好奇心または関連性フック（開封率35%以上が目標）
- **パーソナライゼーション層**: 企業トリガーイベント、役割別ペインポイント、共通のつながり
- **フォローアップタイミング**: 1日目→3日目→7日目→14日目（最低4タッチ）
- **CTA最適化**: 最初はソフトなお願い（「お話しする価値はありますか？」）、フォローアップでハードなお願い

## ウェビナーファネル

教育コンテンツによるハイインテントリード獲得。

- **トピック選定**: ICPが今四半期に直面する具体的な課題を解決
- **プロモーションミックス**: メールリスト（40%）、LinkedIn投稿（30%）、パートナークロスプロモーション（30%）
- **ウェビナー後のナーチャリング**: リプレイ＋キーテイクアウェイメール→ケーススタディ→デモオファーシーケンス
- **コンバージョンベンチマーク**: 登録→出席30〜40%、出席→SQL 10〜15%

## ABMレイヤー

エンタープライズ案件（ACV $50K以上）向けのアカウントベースドターゲティング。

- **ターゲットアカウントリスト**: 最大50〜100アカウント、四半期ごとに更新
- **マルチスレッディング**: 1アカウントにつき3〜5人のステークホルダーに同時アプローチ
- **カスタムコンテンツ**: パーソナライズドLP、ROI計算ツール、業界別ケーススタディ
- **シグナルトラッキング**: インテントデータ、ウェブサイト訪問、コンテンツエンゲージメントスコアリング

## メトリクスダッシュボード

先行指標と遅行指標でパイプラインの健全性を追跡。

- **パイプラインベロシティ**: ディールがステージを通過する速度
- **SQL率**: マーケティング適格→セールス適格への転換率
- **CACベンチマーク**: チャネル別、セグメント別、時系列トレンド
- **レベニューアトリビューション**: 正確なチャネルROIのためのマルチタッチアトリビューションモデル
