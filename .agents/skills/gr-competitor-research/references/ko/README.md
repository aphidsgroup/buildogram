# 경쟁사 조사 & 성장 플라이휠 플레이북 — 완전 SOP

경쟁사가 어떻게 성장했는지 역설계한다: 웹사이트 변천, 소셜 분해, 플라이휠 스코어링, 전파 체인. 150개 이상의 AI 스타트업 분석으로 검증. Lovable 완전 케이스 스터디(2024년 11월 런칭: 229K 임프레션, 영상 430만 뷰, Discord 1만 명 이상) 포함.

## 4단계 조사 프레임워크

- **Step 1 — 공식 사이트 분해**: 단계별 사이트 변천을 추적해 가치 제안(Value Proposition)의 진화와 획득 구조를 산출
- **Step 2 — 소셜 채널 분해**: 채널별 콘텐츠 분류, 게시 리듬, KOL 협업 패턴
- **Step 3 — 디렉터리 사이트 분해**(필요 시): 유입원과 게재 진입점
- **Step 4 — 광고 집행 분해**(필요 시): 집행 채널, 소재 전략, 트래픽 구성
- 도구: Wayback Machine(과거 스냅샷), SimilarWeb(트래픽 분석), Semrush(SEO·광고)

## 웹사이트 변천 — 3개 버전 체크포인트

V1(최소 구성) / Beta(클로즈드 테스트기) / Launch(정식 공개)를 비교. 각 버전마다 기록: 한 줄 소개(Value Prop), 페이지 길이와 구조, 사용자 사례, Banner/Pricing/CTA 프레임, SEO 변화, 커뮤니티 채널 시작 시점, 외부 연동.

## 소셜 분해

- 먼저 SimilarWeb으로 주요 유입원을 파악한 뒤 해당 채널을 깊이 파고든다. **그 채널을 잘하는 경쟁사를 우선 분석할 것**
- **계정 매트릭스**: 공식 계정(빈도·콘텐츠 구성) / 창업자 계정(공식과의 차별화 정도) / DevRel 계정(BD·에코시스템) / 크리에이터 계정(단계별 KOL 협업)
- **게시 리듬 4단계**: 클로즈드 베타 → 대규모 공개 → 장기 운영 → Launch Week
- **플랫폼별 역할**: X/Twitter = 점화(일 1–3건), YouTube = 활성화(주 1–3건), LinkedIn = 엔터프라이즈 획득(주 1–2건), TikTok = 신규 도달

## 성장 플라이휠 — 6단계 스코어링

단계별 채점: 창작/Activation(아하 모먼트를 어떻게 만드나?) → 공유/Referral(제품 내 바이럴 메커니즘은?) → 발견/Acquisition(잠재 사용자는 어떻게 유입되나?) → 상호작용/Retention(커뮤니티 소속감은?) → 수익화/Revenue(무료→유료 경로는?) → 제품 개선(피드백이 이터레이션을 움직이는가?)

| 신호 | 강한 플라이휠 | 약한 플라이휠 |
|------|---------------|---------------|
| 리텐션 | Discord DAU 높고 UGC 지속 | 이탈이 빠름 |
| 입소문 | 사용자 스토리의 체계적 증폭 | 증폭 메커니즘 없음 |
| 결제 | 트리거가 명확 | 경로가 모호 |
| 지속성 | 제품과 커뮤니티의 폐쇄 루프 | 외부 광고 의존 |

## 전파 체인 & KOL 식별

- **4단계 전파 모델**: 내부 점화(발표 후 몇 시간, 팀 일제 발신) → 에코시스템 촉매(1–2일, 파트너 지지) → KOL 폭발(2–3일, 톱 보이스가 평가를 규정) → 광범위 침투(수 주간, 다국어 사용자 참여)
- **콘텐츠 효과 순위(데이터 검증)**: 비주얼 데모 > 실사용자 성공 사례 > 경쟁사 비교 > 정량 데이터 > 기능 공지
- **KOL 3유형**: 트래픽 킹(포스트당 10만 임프레션 초과), 화제 발화점(댓글창의 깊은 토론), 인식 전환점("앞으로는 이것만 쓴다")
- 단일 포스트 분해: Hook(어떤 각도로 말하나) / Nugget(무엇이 공유 욕구를 만드나) / Feeling(감탄·호기심·FOMO) / 비주얼(어떤 이해 비용을 낮추나)

## 리포트 제출 전 셀프 체크

3개 버전 체크포인트를 다뤘는가. 주요 소셜 채널을 모두 분해했는가. 플라이휠의 강약을 판정했는가. 전파 주장에 데이터 근거가 있는가. 재사용 가능한 전술과 적용 경계를 추출했는가. 리스크와 조기 경보 지표를 짚었는가. ICP를 3개 유형 이상 다뤘는가. 가격의 업그레이드 경로를 분석했는가. 실행 가능한 제안이 있는가.
