# スタートアップ成長コンサルタント — PH ローンチ、OSS、GTM 監査

スタートアップの最もレバレッジの高い3つの成長課題に対するオンデマンドコンサルティング。

## Product Hunt 監査

ローンチ前にアセット、タイミング、戦略をレビュー：

- **ギャラリーレビュー**: 最初の画像はプロダクトの動作画面（ロゴではない）。モバイルサムネイルに最適化。
- **タグライン評価**: 60文字以内、ベネフィット重視、専門用語なし。5バージョンをピアでテスト。
- **タイミング分析**: 火曜日（競争激しい）を避ける; 木曜/金曜の競争が弱いフィールドを狙う。
- **Makerコメント戦略**: 脆弱性と具体性のある500語の起源ストーリーを準備。
- **アップボート動員計画**: 当日シーケンス（DM→メール→SNS）を正確なタイミングウィンドウで。

## OSS成長レビュー

リポジトリ、README、ディストリビューションをStar獲得速度の観点で分析：

- **README監査**: ヒーロー画像 + ワンライナー + GIFデモ + クイックスタートを最初の画面内に
- **Star→コントリビューターファネル**: Good first issueラベル、CONTRIBUTING.md、レスポンス24時間以内
- **ディストリビューションチャネル**: Show HN、Reddit、Dev.to、Twitterに投稿しているか、どの順序か？
- **ソーシャルプルーフシグナル**: Starバッジ、コントリビューター数、「used by」ロゴ、READMEの推薦文
- **リリース頻度**: 魅力的なチェンジログ付きの定期リリース（隔週）がStar成長を維持

## GTM戦略評価

ポジショニング、チャネル、メッセージングのフィードバック：

- **ポジショニング監査**: 初見の人が8秒であなたが何をしていて、なぜ重要かを理解できるか？
- **ICPの明確さ**: 1つのビーチヘッドセグメントに絞っているか、全員を喜ばせようとしていないか？
- **チャネルフォーカス**: 5つ以上のチャネルに手を広げ、各々中途半端な結果になっていないか？
- **メッセージングの一貫性**: ホームページ、広告、メール、プロダクトが同じストーリーを語っているか？
- **競合差別化**: 競合に言及せず、独自の価値を一文で伝えられるか？

## スプリントプランニング

明確なデリバラブル付きの優先順位付け2週間グローススプリント：

- **ICEスコアリング**: Impact × Confidence × Easeで実験を評価; スプリントごとにトップ3を実行
- **スプリント構造**: 月（計画+セットアップ）→ 火〜木（実行）→ 金（計測+ドキュメント化）
- **アウトプットテンプレート**: 各スプリントでブログ1本、ディストリビューション1プッシュ、指標1改善
- **レトロスペクティブ**: 何が効いた？何が効かなかった？何に倍賭けするか？
- **複利の原則**: 各スプリントは前のスプリントの学びの上に積み上げる

## フラクショナル成長リード

フルタイム採用なしの継続的な戦略ガイダンス：

- **エンゲージメントモデル**: 週4〜8時間の戦略的インプット + 週次シンクコール
- **スコープ**: チャネル戦略、キャンペーンレビュー、実験設計、採用アドバイス
- **フラクショナルを雇うべき時**: Post-PMF、プレシリーズA、ファウンダーが成長とプロダクト両方できない時
- **デリバラブル**: 月次成長戦略書、週次実験レビュー、四半期プランニング
- **移行計画**: 6〜12ヶ月かけて社内グロース能力を構築し、フルタイム採用が合理的になるまで
