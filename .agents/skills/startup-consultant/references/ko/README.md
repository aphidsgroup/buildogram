# 스타트업 성장 컨설턴트 — PH 론칭, OSS, GTM 감사

스타트업의 가장 레버리지 높은 3가지 성장 과제에 대한 온디맨드 컨설팅입니다.

## Product Hunt 감사

론칭 전 에셋, 타이밍, 전략을 리뷰:

- **갤러리 리뷰**: 첫 이미지는 제품 동작 화면 (로고 아님). 모바일 썸네일에 최적화.
- **태그라인 평가**: 60자 이내, 혜택 중심, 전문 용어 배제. 5개 버전을 동료와 테스트.
- **타이밍 분석**: 화요일(경쟁 치열) 피하기; 목/금의 약한 필드를 노리세요.
- **Maker 코멘트 전략**: 취약성과 구체성이 담긴 500단어 탄생 스토리 준비.
- **업보트 동원 계획**: 당일 시퀀스(DM→이메일→소셜)를 정확한 타이밍 윈도우로.

## OSS 성장 리뷰

리포지토리, README, 배포를 Star 획득 속도 관점에서 분석:

- **README 감사**: 히어로 이미지 + 한줄 설명 + GIF 데모 + 퀵 스타트를 첫 화면에
- **Star→컨트리뷰터 퍼널**: Good first issue 라벨, CONTRIBUTING.md, 응답 시간 24시간 이내
- **배포 채널**: Show HN, Reddit, Dev.to, Twitter에 게시하고 있나요? 어떤 순서로?
- **소셜 프루프 시그널**: Star 뱃지, 컨트리뷰터 수, "used by" 로고, README 내 추천사
- **릴리스 주기**: 매력적인 체인지로그와 함께 정기 릴리스(격주)가 Star 성장을 유지

## GTM 전략 평가

포지셔닝, 채널, 메시징에 대한 피드백:

- **포지셔닝 감사**: 처음 본 사람이 8초 안에 뭘 하는지 왜 중요한지 이해할 수 있나요?
- **ICP 명확성**: 하나의 비치헤드 세그먼트로 좁혔나요? 모두를 만족시키려 하고 있진 않나요?
- **채널 집중**: 5개 이상 채널에 분산되어 각각 미미한 결과를 내고 있지 않나요?
- **메시징 일관성**: 홈페이지, 광고, 이메일, 제품이 같은 스토리를 전하고 있나요?
- **경쟁 차별화**: 경쟁사를 언급하지 않고 독자적 가치를 한 문장으로 전달할 수 있나요?

## 스프린트 플래닝

명확한 산출물이 있는 우선순위 기반 2주 그로스 스프린트:

- **ICE 스코어링**: Impact × Confidence × Ease로 실험 평가; 스프린트당 상위 3개 실행
- **스프린트 구조**: 월(계획+설정) → 화~목(실행) → 금(측정+문서화)
- **아웃풋 템플릿**: 각 스프린트에서 블로그 1편, 배포 1푸시, 지표 1개 개선
- **회고 포맷**: 뭐가 됐는지? 뭐가 안 됐는지? 뭘 더 밀어붙일지?
- **복리의 원칙**: 각 스프린트는 이전 스프린트의 학습 위에 쌓아야 합니다

## 프랙셔널 그로스 리드

풀타임 채용 없는 지속적 전략 가이던스:

- **인게이지먼트 모델**: 주 4~8시간 전략적 인풋 + 주간 싱크 콜
- **범위**: 채널 전략, 캠페인 리뷰, 실험 설계, 채용 가이던스
- **프랙셔널 고용 시점**: Post-PMF, 프리 시리즈A, 파운더가 성장과 제품 둘 다 할 수 없을 때
- **산출물**: 월간 성장 전략서, 주간 실험 리뷰, 분기별 플래닝
- **전환 계획**: 6~12개월에 걸쳐 내부 그로스 역량을 구축하고 풀타임 채용이 합리적일 때까지
