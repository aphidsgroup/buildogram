# デベロッパーマーケティングプレイブック — DevRel、ドキュメント & コミュニティファネル

エンジニアは従来型マーケティングには反応しません。本当に効くのはこれです。

## DevRelプログラム設計

アドプションを促進するロール、メトリクス、アクティビティ：

- **コアロール**: デベロッパーアドボケイト（コンテンツ+コミュニティ）、DXエンジニア（DX+SDK）、コミュニティマネージャー（エンゲージメント+サポート）
- **主要メトリクス**: 初回Hello Worldまでの時間、ドキュメントPV、デベロッパーあたりAPIコール数、コミュニティ成長率
- **アクティビティマトリクス**: カンファレンス登壇（認知）、チュートリアル（アクティベーション）、オフィスアワー（リテンション）、ハッカソン（拡大）
- **採用プロファイル**: コードが書けてかつコミュニケーションできる元デベロッパー; 純粋なマーケターは避ける
- **予算配分**: 40% イベント/出張、30% コンテンツ制作、20% ツール/インフラ、10% スワッグ

## ドキュメントを成長エンジンに

優れたドキュメントが最高のユーザー獲得チャネルになる仕組み：

- **SEOの金鉱**: 各APIエンドポイント = 「[アクション] + [言語]」クエリの潜在ランディングページ
- **インタラクティブな例**: ドキュメント内で実行可能なコード（CodeSandbox、Replit埋め込み）→ エンゲージメント3倍
- **段階的開示**: クイックスタート（5分）→ チュートリアル（30分）→ リファレンス（網羅的）
- **コントリビューションモデル**: コミュニティがGitHub PRでドキュメント改善 → オーナーシップ → アドボカシー
- **メトリクス**: ドキュメント→サインアップ転換率、滞在時間、検索→成功率

## API開発者体験

初回APIコールまでの時間を短縮するDX改善：

- **ゼロコンフィグオンボーディング**: 自動生成APIキー、プリフィル例、コピペ可能
- **SDK品質**: 初日からトップ5言語をサポート; OpenAPI仕様から自動生成
- **エラーメッセージ**: 修正提案と関連ドキュメントリンク付きの人間が読めるエラー
- **プレイグラウンド/サンドボックス**: コミットする前にテストできるインタラクティブAPIエクスプローラー
- **チェンジログ規律**: セマンティックバージョニング、マイグレーションガイド、破壊的変更は90日前に予告

## コミュニティファネル

Discord/Slack → コントリビューター → チャンピオン → カスタマーパイプライン：

- **ステージ1 — ロム専**: サーバーに参加、メッセージを読む、コンテンツを消費
- **ステージ2 — 参加者**: 質問する、他者に回答、プロジェクトを共有
- **ステージ3 — コントリビューター**: PRを提出、チュートリアル執筆、質の高いバグ報告
- **ステージ4 — チャンピオン**: 外部でアドボケイト、登壇、新参者のメンタリング
- **ステージ5 — カスタマー**: チーム/会社が採用した時に有料転換; 社内バイインをチャンピオン

## ハッカソンROI

いつハッカソンが有効で、どう運営し、インパクトを測定するか：

- **スポンサーすべき時**: プロダクトがデベロッパーがビルドするプラットフォームである場合のみ
- **フォーマット**: スターターテンプレート提供、専用メンタートラック、最優秀インテグレーション賞
- **測定**: API利用プロジェクト数、ハッカソン後リテンション（30日間のAPIコール）
- **フォローアップ**: トッププロジェクトをブログで紹介、延長クレジット提供、チャンピオンプログラムに招待
- **コスト目安**: オンライン $5K〜$20K; 対面100名以上 $20K〜$100K
