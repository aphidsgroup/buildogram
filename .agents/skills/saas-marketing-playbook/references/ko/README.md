# SaaS 마케팅 스택 — 도구 + 채널 + 지표

어떤 마케팅 도구를 구매해야 할지 고민하지 마세요. 이 플레이북은 단계에 맞는 스택을 매핑합니다.

## 도구 추천

애널리틱스, 어트리뷰션, 이메일, CRM — 단계와 예산에 맞춰.

- **$0~$1K MRR**: 무료 도구 — PostHog, Brevo, Notion CRM, Google Analytics
- **$1K~$10K MRR**: 스타터 유료 플랜 — Mixpanel, ConvertKit, HubSpot Starter, Hotjar
- **$10K+ MRR**: 그로스 스택 — Amplitude, Customer.io, Salesforce/HubSpot Pro, Clearbit
- **원칙**: 과잉 구매 금지; "필요할지도" 모를 때가 아니라 실제로 한계에 도달했을 때 업그레이드

## 채널 우선순위

각 지출 수준에서 작동하는 채널.

- **$0/월 (오가닉만)**: SEO 콘텐츠, 커뮤니티 참여, 리퍼럴 프로그램, Product Hunt
- **$10K/월**: 구글 광고(브랜드 + 경쟁사), LinkedIn 광고(소규모 ICP 오디언스) 추가
- **$100K/월**: Facebook/Instagram 리타겟팅, 팟캐스트 스폰서십, 이벤트로 확장
- **프레임워크**: CAC, 결과까지의 시간, 확장성, 창업자 핏으로 채널을 스코어링

## 콘텐츠 마케팅 엔진

전략적 콘텐츠를 통한 지속 가능한 오가닉 성장.

- **SEO 기반**: 카테고리 페이지, 비교 페이지("X vs Y"), 대안 페이지
- **블로그 전략**: 상업적 의도가 있는 롱테일 키워드를 타겟으로 월 2~4편
- **배포**: LinkedIn, 트위터 스레드, 뉴스레터, 신디케이션 플랫폼으로 재활용
- **내부 링크**: 필러 페이지와 클러스터 콘텐츠를 연결하는 허브 앤 스포크 모델

## 유료 광고

언제 시작하고, 어떻게 테스트하고, 언제 중단할 것인가.

- **시작 시그널**: 오가닉 CAC 확립, 전환 퍼널 검증 완료, LTV > 3x 목표 CAC
- **테스트 프레임워크**: 채널 테스트당 $500~$1K, 최소 2주, 명확한 성공 기준
- **중단 기준**: 4주 최적화 후에도 CAC > 2x 목표면 다음으로 이동
- **스케일 시그널**: CAC가 목표에서 안정, 양의 ROAS, 예산 증가 시 CAC 급등 없음

## 지표 프레임워크

노스 스타 메트릭, 선행 지표, 리포팅 리듬.

- **노스 스타**: 순 신규 MRR (획득 + 확장 - 이탈 합산)
- **선행 지표**: 가입 수, 활성화율, PQL 수, 트라이얼→유료 전환율
- **주간 리포팅**: 채널별 CAC, 파이프라인 생성량, 단계별 전환율
- **월간 리뷰**: 코호트 리텐션, LTV 트렌드, 채널 믹스 최적화
