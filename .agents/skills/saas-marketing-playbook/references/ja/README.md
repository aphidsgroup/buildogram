# SaaSマーケティングスタック — ツール + チャネル + メトリクス

どのマーケティングツールを買うべきか迷わない。このプレイブックはステージに合ったスタックをマッピング。

## ツール推奨

アナリティクス、アトリビューション、メール、CRM — ステージと予算に合わせて。

- **$0〜$1K MRR**: 無料枠ツール — PostHog、Brevo、Notion CRM、Google Analytics
- **$1K〜$10K MRR**: スタータープラン — Mixpanel、ConvertKit、HubSpot Starter、Hotjar
- **$10K+ MRR**: グローススタック — Amplitude、Customer.io、Salesforce/HubSpot Pro、Clearbit
- **原則**: 過剰購入しない。「必要になるかも」ではなく、実際に上限に達したらアップグレード

## チャネル優先順位

各支出レベルで機能するチャネル。

- **$0/月（オーガニックのみ）**: SEOコンテンツ、コミュニティ参加、リファラルプログラム、Product Hunt
- **$10K/月**: Google広告（ブランド＋競合）、LinkedIn広告（小規模ICPオーディエンス）を追加
- **$100K/月**: Facebook/Instagramリターゲティング、ポッドキャストスポンサーシップ、イベントへ拡大
- **フレームワーク**: CAC、結果までの時間、スケーラビリティ、創業者フィットでチャネルをスコアリング

## コンテンツマーケティングエンジン

戦略的コンテンツによる持続可能なオーガニック成長。

- **SEO基盤**: カテゴリページ、比較ページ（「X vs Y」）、オルタナティブページ
- **ブログ戦略**: 商業意図のあるロングテールキーワードを狙い月2〜4記事
- **ディストリビューション**: LinkedIn、Twitterスレッド、ニュースレター、シンジケーションプラットフォームへ再利用
- **内部リンク**: ピラーページとクラスターコンテンツを繋ぐハブ＆スポークモデル

## 有料広告

いつ始めるか、どうテストするか、いつ止めるか。

- **開始シグナル**: オーガニックCACが確立、コンバージョンファネルが実証済み、LTV > 3x 目標CAC
- **テストフレームワーク**: チャネルテストあたり$500〜$1K、最低2週間、明確な成功基準
- **停止基準**: 4週間の最適化後もCAC > 2x 目標なら次へ
- **スケールシグナル**: CACが目標で安定、正のROAS、予算増加でCACが急上昇しない

## メトリクスフレームワーク

ノーススターメトリクス、先行指標、レポーティングリズム。

- **ノーススター**: ネット新規MRR（獲得＋拡張−チャーンの合計）
- **先行指標**: サインアップ数、アクティベーション率、PQL数、トライアル→有料転換率
- **週次レポーティング**: チャネル別CAC、パイプライン生成量、ステージ別転換率
- **月次レビュー**: コホートリテンション、LTVトレンド、チャネルミックス最適化
