# GEO 证据层标准（证据优先，2026-06）

> 一句话：**AI 引用最看「证据」。** 结构做满只是及格线，真正拉开引用率的是权威引语 + 完整统计 + 可核验出处。
> 来源：一手论文（Aggarwal KDD'24）+ 姚金刚 GEO 权重法（MIT，署名）+ AFFiNE/analook 实践。

## 二、证据纪律（永不编造，标注出处）

每条增强按可信度打标：
- `[原文支持]` —— 原文/我们自己可验证的数据（AFFiNE 60K stars、analook 32K 曝光）
- `[外部已核验]` —— 引一手源，**先核实再写**（论文/官方报告/权威机构）
- `[建议补充]` —— 查不到的**标注待补，绝不瞎编**

来源优先级：**学术论文 > 行业报告 > 权威机构 > 专家观点 > 实践案例**。

## 四、写一篇 GEO 友好内容的执行清单

- [ ] 每个核心结论：配 1 条 `[外部已核验]` 权威引语 **或** 1 个带全口径统计
- [ ] 关键统计写全：数值 + 样本 + 周期 + 来源（例："30 天 / 120 篇 / +32% / 来源 X"）
- [ ] 至少 1 处引一手论文（用上面引用库）
- [ ] 作者署名 + 履历（E-E-A-T）+ 适用边界
- [ ] 过一遍惩罚项：无堆砌、无编造
- [ ] 结构层照旧：摘要 + FAQ + JSON-LD（Article + FAQPage）
