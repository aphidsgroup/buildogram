## English

### What's Inside

This playbook covers **sustained GitHub star growth** — the repeatable monthly cadence that takes a project from post-launch to 10K, 50K, and beyond.

**Core Framework — 300 Stars/Month Cadence:**

1. **Content Rhythm** — Weekly blog posts, YouTube tutorials, daily social media
2. **Community Operations** — Reddit, Hacker News, Twitter/X, LinkedIn strategy
3. **Event-Driven Growth** — Monthly online events, quarterly offline meetups
4. **Contributor System** — Good First Issues → Regular → Core → Maintainer pipeline
5. **Ambassador Program** — Community advocates with incentive structure

### Quick Start

```bash
# Install as AI agent skill
clawhub install gingiris-github-star-growth

# Or clone directly
git clone https://github.com/Gingiris-1031/gingiris-github-star-growth.git
```

### Monthly Execution Checklist

**Daily:**
- [ ] Twitter/X: 1-2 technical posts
- [ ] KOL DM outreach: 3-4 people
- [ ] Issue/PR response: < 24h

**Weekly:**
- [ ] Reddit: 2-3 quality posts (r/selfhosted, r/programming, etc.)
- [ ] LinkedIn: 1 long-form article
- [ ] YouTube: 1 tutorial/demo video
- [ ] Blog: 2-3 technical articles (Dev.to / Medium)
- [ ] Good First Issues: label 3-5 new ones

**Monthly:**
- [ ] Hacker News: 1-2 submissions (Agency: $500/post → 30 points)
- [ ] Online event: 1 webinar/AMA (target: 31-34 attendees)
- [ ] Contributor spotlight: highlight top contributors
- [ ] Growth review: star velocity, content performance, channel ROI

**Quarterly:**
- [ ] Offline meetup (target: 70 people)
- [ ] PR Challenge / Hacktoberfest sprint
- [ ] Ambassador recruitment round
- [ ] Strategy review & channel rebalancing

## FAQ

### Q1: How is this different from gingiris-opensource?
**A:** `gingiris-opensource` is for **launch** — getting your first 1,000 stars in 72 hours. This playbook is for **sustained growth** — maintaining 300+ stars/month after launch, scaling to 10K, 50K, 60K+.

### Q2: What's the minimum budget needed?
**A:** You can start with $0 (pure organic: Reddit + Twitter + Dev.to). Adding $500-1000/month for HN Agency + YouTube ads significantly accelerates growth.

### Q3: How long until I see results?
**A:** With consistent execution of the monthly SOP, expect to hit 300 stars/month cadence within 2-3 months. Client Project A achieved 10K stars in 8 months using this exact framework.

### Q4: Does this work for non-AI projects?
**A:** Yes. The framework is channel-agnostic. AFFiNE (productivity tool) and Client Project A (AI) both used this SOP. Adjust subreddit targets and content angles for your niche.

### Q5: How important is the Contributor system?
**A:** Critical for projects targeting 10K+. Contributors create a virtuous cycle: they star, they evangelize, they bring their networks. AFFiNE's contributor ecosystem was a major growth driver.

### Q6: What's the ROI of HN Agency ($500/post)?
**A:** At $500/post targeting 30+ points, a successful HN post can bring 200-500 stars in 24 hours. ROI depends on your Star→User→Revenue conversion, but for awareness it's one of the highest-leverage channels.

### Q7: Should I do online or offline events first?
**A:** Start with monthly online events (lower cost, 31-34 attendees). Add quarterly offline meetups once you have local community density (target: 70 people). Japan is particularly effective for offline events due to high developer community engagement.

### Q8: How do I measure if the SOP is working?
**A:** Track star velocity (stars/week), content engagement (views, shares), contributor PRs/month, and event attendance. If you're hitting 300 stars/month consistently, the system is working.

### Q9: Can I use this with AI agents?
**A:** Yes! Install via `clawhub install gingiris-github-star-growth` and your AI agent can help draft content calendars, plan outreach sequences, and generate monthly growth reports.

### Q10: What tools do I need?
**A:** Phantombuster (LinkedIn/Twitter DM automation, $49-200/month), YouTube Ads (30 SGD minimum), HN Agency ($500/post), and standard analytics (GitHub Insights, Google Analytics).
