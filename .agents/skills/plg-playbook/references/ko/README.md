# PLG(제품 주도 성장) 플레이북 — 프리미엄에서 엔터프라이즈까지

제품이 스스로 영업하게 하라. PLG 모션의 3단계로 구성.

## 프리미엄 설계

- **'아하 모먼트'는 무료로 제공**: 무료 티어는 핵심 가치를 경험할 수 있어야 하며, 반쪽짜리 데모가 되어서는 안 됨
- **가치가 아닌 규모에 과금**: 시트 수·사용량·관리 기능에 과금하고, 사용자가 사랑하는 기능에는 과금하지 않기
- **가격 심리학**: 아직 제공 전이라도 엔터프라이즈 티어를 보여줘 앵커로 활용

## 셀프서브 온보딩

- **활성화 정의는 행동 임계값으로**(예: 실제 작업 2건 이상 완료). '가입 완료'는 정의가 아님
- **Time-to-value 계측**: 가입→활성화 전 단계를 계측하고 기여하지 않는 단계는 제거
- **이탈 트리거**: 두 번째 세션 후 24시간 내 재방문 없음 → 자동 윈백 이메일
- **결제 어트리뷰션**: 가입 × 활성화 × 결제 × 최근 로그인 4개 타임스탬프

## 엔터프라이즈 핸드오프(PLG → SLG)

- **시트 수 게이트**: 커스텀/프라이빗 배포는 N시트 이상만
- **프라이빗 배포의 진짜 이유는 두 가지뿐**: 데이터 프라이버시 + 사내 권한 시스템 연동. 그 외에는 SaaS로 유도
- **계약 규율**: 분할 결제(3:4:3), SOW는 계약과 동시 서명, 변경은 CCB 경유 — 서면만 유효
- **리소스 배분**: SLG 주도 회사도 약 20% 리소스를 PLG에 남겨 파이프라인을 키움

## 벤치마크

| 지표 | 건강한 PLG |
|------|-----------|
| 가입 → 유료 | >5% (Pro-C 5–8%, SMB 10–15%) |
| 활성화율 | 40%+ (초기 단계 하한 20%) |
| 온보딩 주간 리텐션 | >25–30% |
