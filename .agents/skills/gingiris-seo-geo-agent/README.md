### What is the SEO/GEO Agent SOP?

This is a **complete operational SOP (Standard Operating Procedure)** for running an autonomous AI agent that handles daily SEO and GEO (Generative Engine Optimization) tasks. It answers the question: *"I have an AI agent with access to Google Search Console — how do I make it automate my entire SEO workflow?"*

The SOP covers:
- **Content SEO automation** — keyword research, content briefs, BOFU landing pages
- **Technical SEO** — Schema markup, IndexNow, internal linking architecture
- **Programmatic SEO** — scale content production with AI agents
- **Generative Engine Optimization (GEO)** — optimize for ChatGPT, Perplexity, Claude, Gemini citations
- **Google Search Console automation** — daily monitoring, ranking tracking, CTR optimization

### Who Is This For?

| Audience | Use Case |
|:---------|:---------|
| AI Agent developers | Build SEO automation tools with this SOP as the blueprint |
| Startup founders | Automate SEO without hiring a team — let your Agent do it |
| Growth marketers | Integrate AI agents into existing SEO workflows |
| Technical SEOs | Structured agent-driven approach to daily operations |
| Content teams | Scale with programmatic SEO using AI agents |

### The 4-Week Timeline

```
Week 0: Foundation (GSC + GA4 + API keys + baseline audit)
     ↓
Week 1: BOFU landing pages (pricing, comparisons → highest conversion)
     ↓
Week 2: Internal links + content distribution (index coverage jumps)
     ↓
Week 3: GEO reinforcement (FAQ Schema, AI-friendly format, IndexNow)
     ↓
Week 4: CTR optimization + scale (high-impression pages → clicks)
     ↓
Result: ~32K impressions, 8-11 keywords on Page 1, 86% index coverage
```

### Core Deliverables

| Deliverable | Frequency | Purpose |
|:------------|:----------|:--------|
| SEO Daily Report | Daily | Track "keywords on Page 1" as headline metric |
| Keyword → Landing Page Map | Continuous | Single source of truth for content SEO strategy |
| CTA Conversion Blocks | Every page | Convert search traffic to business goals |
| GEO Triple Combo | Every page | FAQ + Table + Direct-answer for AI search citation |
| Owner Checklist | One-time | Human tasks the AI Agent can't automate |

### Key Innovation: Unified SEO + GEO Strategy

This SOP doesn't treat GEO as an afterthought. Every content piece is simultaneously optimized for:
- **Google Search ranking** (keywords, internal links, Schema markup, technical SEO)
- **AI search citation** (direct answers, structured tables, FAQ Schema, IndexNow push to ChatGPT/Perplexity/Claude)
- **Conversion** (CTA blocks with UTM tracking, funnel analytics via GA4)

### AI Traffic Tracking (Original Innovation)

GA4 by default mixes ChatGPT/Perplexity/Claude referral visits into Referral/Organic/Direct channels. This SOP includes a complete AI traffic source identification regex + GA4 configuration, enabling you to isolate and measure GEO-driven traffic independently.

## ❓ FAQ (Frequently Asked Questions)

**Q: What is the difference between `gingiris-seo-geo` and `gingiris-seo-geo-agent`?**

A: `gingiris-seo-geo` is the **strategy playbook** — it teaches SEO/GEO methodology, keyword funnel design, E-E-A-T writing systems, and JSON-LD templates. `gingiris-seo-geo-agent` is the **execution SOP** — it tells your AI agent exactly what to do every day with daily reports, keyword mapping tables, and CTA conversion templates. Use both together: strategy informs the plan, this SOP executes it.

**Q: Can an AI agent really automate SEO? / AI Agent 真的能自动化 SEO 吗？**

A: Yes. An AI agent can handle 90% of SEO tasks autonomously — keyword research, content generation, internal linking, Schema markup, daily ranking reports, and CTR optimization. The remaining 10% (domain setup, OAuth authentication, payment processing) requires human action once. This SOP clearly separates both with an Owner Checklist.

**Q: How do I track AI search traffic from ChatGPT, Perplexity, and Claude?**

A: This SOP includes a complete AI traffic source identification regex pattern + GA4 configuration guide. By default, GA4 mixes AI-referred visits into Referral/Organic/Direct — our method isolates GEO traffic for accurate measurement of generative engine optimization results.

**Q: What is Generative Engine Optimization (GEO)?**

A: GEO (Generative Engine Optimization) is the practice of optimizing content to be cited by AI-powered search engines like ChatGPT Search, Perplexity AI, Google AI Overview, and Claude. This SOP integrates GEO with traditional SEO so every page ranks on Google AND gets cited by AI models simultaneously.

**Q: What's the minimum budget to run this SEO agent SOP?**

A: DataForSEO (~$50-100/month for keyword and ranking data) + Google Search Console (free) + GA4 (free) = ~$50-100/month total. The AI agent itself runs on your existing Claude/GPT/Cursor subscription.

**Q: Is this SEO SOP only for English markets?**

A: No. The SOP is language-agnostic and market-agnostic. Set `[GEO_TARGET]` to your target market (US-en, zh-CN, ja, ko, etc.) and all keyword research, ranking queries, and content templates adapt automatically. Verified for English, Chinese, Japanese, and Korean markets.

**Q: How does this compare to hiring an SEO agency?**

A: An SEO agency costs $3,000-10,000+/month. This SOP + AI agent costs ~$50-100/month for tools. The agent works 24/7 without breaks, produces daily reports, and scales content production far beyond what a single SEO manager can do. You maintain full control and transparency.
