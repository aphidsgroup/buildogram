# 小红书笔记模板

## 标题公式（从 75 个爆款公式抽的 5 类，高频适合 SaaS / 出海博客拆条）

### 类型 1：数字 + 实测
- `我实测了 27 个 XX 工具，免费能打的只有 4 个`
- `花了 3 万块试错，终于找到这 5 个真香工具`
- `90% 的人还不知道：这个 XX 免费版就够用`

### 类型 2：身份 + 踩坑
- `前 AFFiNE COO：别再这么做 XX 了（血泪教训）`
- `做独立开发 2 年，我终于搞懂了 XX 这件事`
- `从 0 到 PMF，我踩过的 5 个最蠢的坑`

### 类型 3：对比 + 反差
- `用了 3 年 XX，直到我发现 Y 才真香`
- `别被大厂骗了 —— 开源版其实更好用`
- `$3500/月 vs $0/月：付费工具真的值吗`

### 类型 4：疑问 + 拆解
- `为什么 80% 的工具选错了？（附自查清单）`
- `看完再决定要不要付费：XX 工具 30 天实测`
- `到底该选免费还是付费？这张表告诉你答案`

### 类型 5：清单 + 速成
- `5 步搞定 XX（0 基础也能上手）`
- `手把手教你做 XX，附工具清单`
- `新手必看：XX 从 0 到 1 完整指南`

## 内容结构（< 1000 字）

### 开头 100 字（钩子 + 痛点）
```
想分享一个血泪教训。

上个月我花了 3 个月测 27 个社媒监听工具，
前两周就氪了 $400，后来发现其中一半根本不值。

如果你也在纠结要不要付费，这篇可以帮你少走 3 个月弯路。
```

### 正文 600 字（3-5 要点，每个 100-200 字）

**要点 1**：
```
✅ 重点一：先看你的品牌月均提及量

如果你的品牌每月被提及 < 1500 次，
付费工具 90% 是浪费钱。

免费工具 + Reddit/HN 手动搜，就能覆盖 90% 的场景。

我现在还在用的免费组合：
- Toolify AI（500 mentions/月免费）
- Tweet Binder（X 专用）
- PH Deck（Product Hunt 监测）
- Google Alerts

成本：$0/月。
```

**要点 2**、**要点 3**…（同结构，每点独立）

### 结尾 100 字（TL;DR + 互动）
```
TL;DR:
初创团队 < 1500 提及/月，免费工具就够。
多语言（日韩中）监听 = 唯一值得付费的场景。

你们现在用什么监听工具？
评论区聊聊👇

主页简介有完整对比表（30+ 工具测评）
```

## Tag 策略（5-10 个）

**必选（出海 / SaaS 类）**：
`#出海创业` `#独立开发者` `#SaaS` `#创业工具` `#AI工具`

**主题相关（根据博客内容调）**：
- 如 SEO 类：`#SEO优化` `#GEO` `#谷歌SEO`
- 如 PH 类：`#ProductHunt` `#产品发布` `#创业`
- 如 OSS 类：`#开源项目` `#GitHub` `#程序员`
- 如 B2B 类：`#B2B` `#企业服务` `#PMF`

**流量 tag**：`#干货分享` `#真实分享` `#避坑指南`（能带一波自然流量但别堆太多）

## 图片设计 5 张

### 图 1：标题图（封面决定点击率 80%）
- 横屏 3:4 比例
- 大字标题（30-40px）：分两行，第一行数字 / 悬念 / 反差
- 副标题 1 行：说清讲什么
- 右下角：Iris 头像 + "@Gingiris" 水印
- 背景：简洁（白 / 米色），配色 ≤ 3 种

示例：
```

━━━━━━━━━━━━━━━━
@Gingiris · 出海增长实战
```

### 图 2-4：每张一个要点
- 截图 / 表格截图（不是 AI 生成的假图）
- 用红色框 / 箭头标出关键点
- 底部留 1 行备注"这里是重点"

### 图 5（可选）：数据表
- 对比表（3-5 列，5-10 行）
- 优先展示博客里的 Citable Statistics

## 反模式

- ❌ 标题全大写 / 全感叹号（平台会降权）
- ❌ 第一张图用 AI 生成（识别率高，显业余）
- ❌ 正文贴 URL（严重限流 —— 改成"主页简介"）
- ❌ 买数据 / 互赞互评（被检测后 7 天限流）
- ❌ 每天发 3+ 篇（平台会认为是营销号）
- ❌ 全文没互动问题（评论数决定二次推送）

---

## 发布节奏

- **时间**：周三 18-22 点 或 周六 10-14 点
- **频率**：一周 2-3 篇上限
- **tag 用量**：5-8 个（多了像营销号）
- **发布 24h 内**：积极回评论，带动算法推流
