# X Thread 模板库

## Hook 句型 10 种（前 3 推决定 70% 留存）

### 数字派
```
Here's what [N data points] taught me about [topic]:
(thread 🧵)
```
例：`Here's what analyzing 27 social listening tools taught me:`

### 反常识派
```
Everyone says [conventional wisdom].
I tested it for [time period].
Here's what actually worked:
```

### 失败学派
```
I spent $[amount] on [tool/tactic].
It didn't work.
Here's what did (in 13 tweets):
```

### Before/After 派
```
[Month] ago: [painful status]
Today: [current result]
What changed (thread):
```

### 问题派
```
Why does [X] fail for 80% of startups?
I analyzed [data set].
The answer surprised me.
```

## 完整 thread 样例（基于 social listening post）

```
1/ I tested 27 social listening tools for startups.
Only 4 are actually free and useful.
Here's the breakdown (🧵):

2/ Stakes: I spent 3 months switching between tools
Lost ~$400 on two "startup-friendly" plans that weren't
The lesson: match tool tier to mentions/mo

3/ Rule #1:
<1500 brand mentions/mo? → stay free.
Free tier + Reddit/HN search = 90% of paid tool value
Most startups never need more.

4/ Free tier winners (tested):
- Toolify (AI-powered, 500 mentions/mo free)
- Tweet Binder (X-only, great for launch week)
- PH Deck (Product Hunt-specific, free)
- Google Alerts (old school, still works)

5/ Paid tier: only 4 categories worth the money
- Multilingual (JA/KO/ZH): Brandwatch, Sprinklr
- Trend detection: Vizzylabs
- Reddit/HN focus: none worth $79+/mo — manual beats tools here

6/ Prices from my 2026 audit:
- Median startup tier: $79/mo
- Most expensive: Brandwatch ($3,500/mo for consumer intel)
- Cheapest full-featured: $0 (Toolify free + manual)

7/ Multilingual is the real paid-tool differentiator.
Only 6/27 tools handle Japanese/Korean/Chinese well.
If you're going global, this is where paid starts to justify cost.

8/ Channels cheat sheet:
✅ Clean signal: Reddit, HN, LinkedIn (B2B)
⚠️ High noise: TikTok, Instagram Reels
🚫 Not worth tracking as indie: Facebook (2026 data is dead)

9/ 35% of startups drop paid tools in first 90 days.
Why? They didn't validate mentions/mo BEFORE signing up.
Check your mentions/mo first. Use free tier for 30 days. Then decide.

10/ My actual stack right now:
- Toolify (free) for general monitoring
- Reddit saved-search for community signal
- PH Deck for launch tracking
- X native search for customer voice
Cost: $0/mo. Works for <1500 mentions/mo.

11/ When paid tools DO pay off:
- You're enterprise B2B ($10k+ ACV)
- You need multilingual
- You have >1500 mentions/mo (real brand, not hoping for it)

12/ When they DON'T:
- You're pre-PMF
- You're monitoring "my product name" with <100 mentions/mo
- You're doing competitor tracking for 2-3 companies (manual faster)

13/ TL;DR:
For startups <1500 mentions/mo, free tier + Reddit/HN manual monitoring beats $79+ paid tools.
Multilingual is the only reason to pay early.

14/ Full breakdown with pricing table, free tier limits, and multilingual comparison:

→ https://gingiris.tools/blog/2026/04/02/best-social-media-listening-tools-startups-2026/

15/ If this helped, I write weekly about indie founder SEO + AI distribution.
Follow @WeiYipei for more.
```

## 检查清单（发推前）

- [ ] 每推独立可理解（截图单推不丢上下文）
- [ ] Hook 测试：只看推 1，你会继续读吗？
- [ ] 数字不模糊（"27" 比 "许多" 好）
- [ ] 截图 / 表格 ≥ 2 张
- [ ] 最后一推有 CTA + @ 自己
- [ ] UTM 加到 link：`?utm_source=x&utm_medium=social&utm_campaign=distill_{slug}`
