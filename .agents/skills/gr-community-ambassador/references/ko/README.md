# 커뮤니티 & 앰배서더 프로그램 플레이북 — 완전 SOP

사전 체크리스트부터 모집, 단계별 관리, 리텐션, 거버넌스까지. Notion(2,000만 사용자, 앰배서더 실제 인터뷰), AFFiNE(60K 스타), Asana Ambassador, ClickUp Verified 사례에서 구축.

## 핵심 전제

- **목표는 '상호 성취'**: 사용자는 개인 영향력과 수익화 경로를 얻고, 제품은 커뮤니티 주도의 글로벌 확장을 얻는다 — 사용자·콘텐츠·제품 플라이휠
- **앰배서더의 본질**: 배지를 단 홍보 요원이 아니다. 진짜 동기는 KOL이 될 기회, 창업자와의 직접 대화, 수익화 경로
- **운영 철학**: 장기 동행과 함께 성장하기. 일회성 트래픽 수확은 하지 않는다

## 시작 전 — 6가지 조건 모두 충족

- 팀 전원이 자사 제품을 쓰고, 일상 워크플로에 녹아 있을 것
- 제품이 진짜 '쓸 만한' 상태에 도달했을 것
- 콘텐츠 수용 역량, 공개 가능한 로드맵, 전담 담당자가 있을 것
- **단계 적합성**: PMF 이전에는 시작하지 말 것(먼저 딥 유저 100명 찾기). 성장기에는 시드 앰배서더 5–10명을 할당량 없이. 확장기에 정식 프로그램 + 포인트제, 성숙기에는 풀 에코시스템 + 제휴 프로그램

## 모집과 등급 체계

- **가입 기준(A와 B 동시 충족)**: A — 이벤트 운영 능력, B — 콘텐츠 제작 능력(영상·글·템플릿 중 하나)
- 양보다 질: Notion 명단은 수백 명 규모지만 실제 활동은 약 25%
- **승급 체계**: Bronze(신청 즉시) → Silver(연 100점 이상, 베타 우선 + 공식 리트윗) → Gold(300점 이상, 1:1 담당 + 연례 행사) → Platinum(600점 이상, 맞춤 지원 + 스폰서십)
- 포인트 예시: 튜토리얼 장문 20, 영상 30, 로컬 이벤트 50, 버그 리포트 10, 문서 번역 15, 채택된 커뮤니티 답변 3

## 5대 인센티브 기둥 (Notion 인사이트)

1. 제품이 진짜 좋을 것 — 모든 것의 전제 조건
2. 베타 우선 접근 — '인사이더'라는 감각
3. 창업자와의 직접 대화 — 가장 강력한 인센티브
4. KOL로 성장하도록 지원 — 공식 리트윗과 어나운스
5. 모든 크리에이터를 리트윗 — 언급한 모두를 격려하기

## 이탈 조기 경보

- 2개월간 기여 없음 → 개인 DM으로 안부 확인
- 품질 하락 → 트레이닝 리소스 제공. '존중받지 못한다'는 느낌 → 1:1 + 확산 지원
