# コミュニティ＆アンバサダープログラム・プレイブック — 完全 SOP

事前チェックリストから採用、段階管理、リテンション、ガバナンスまで。Notion（2,000 万ユーザー、アンバサダーへの実インタビュー）、AFFiNE（60K スター）、Asana Ambassador、ClickUp Verified の実例から構築。

## 前提となる考え方

- **目標は「相互成就」**：ユーザーは個人の影響力とマネタイズへの道を得て、プロダクトはコミュニティ主導のグローバル拡大を得る——ユーザー・コンテンツ・プロダクトのフライホイール
- **アンバサダーの本質**：バッジを付けた宣伝係ではない。本当の動機は、KOL になれる機会・創業者との直接対話・マネタイズへの道筋
- **運営理念**：長期的な伴走と共同成長。一度きりのトラフィック刈り取りはしない

## 立ち上げ前——6 条件すべてを満たすこと

- チーム全員が自社プロダクトを使い、日常のワークフローに組み込まれている
- プロダクトが本当に「使える」状態に達している
- コンテンツを受け止める体制・公開できるロードマップ・専任担当者がある
- **フェーズ適合**：PMF 前は開始しない（まず 100 人の深いユーザーを見つける）。成長期はシードアンバサダー 5–10 名をノルマなしで。拡大期に正式プログラム＋ポイント制、成熟期はフルエコシステム＋アフィリエイト

## 採用と段階制度

- **参加基準（A と B を同時に満たす）**：A＝イベント運営力、B＝コンテンツ制作力（動画・執筆・テンプレートのいずれか）
- 質＞量：Notion の名簿は数百人規模だが、実際に活動しているのは約 25%
- **昇格制度**：Bronze（申請で付与）→ Silver（年 100 pt 以上、ベータ優先＋公式リツイート）→ Gold（300 pt 以上、1on1 対応＋年次イベント）→ Platinum（600 pt 以上、カスタム支援＋スポンサー）
- ポイント例：チュートリアル長文 20、動画 30、ローカルイベント 50、バグ報告 10、ドキュメント翻訳 15、採用されたコミュニティ回答 3

## 5 つのインセンティブ柱（Notion の実践知）

1. プロダクトが本当に良い——すべての前提条件
2. ベータ優先アクセス——「インサイダー」の感覚
3. 創業者との直接対話——最強のインセンティブ
4. KOL への成長を後押し——公式リツイートとアナウンス
5. すべてのクリエイターをリツイート——言及した全員を励ます

## 離脱の早期警戒

- 2 か月間貢献なし → 個別 DM でケア
- 品質低下 → トレーニングリソース提供。「重視されていない」感 → 1on1 ＋拡散支援
