# SaaS 성장 플레이북 — MRR $0에서 $50K까지

매출이 유일한 지표입니다. 이 플레이북은 MRR 마일스톤별로 정리되어 있습니다.

## $0~$1K MRR: 첫 유료 고객

더 개발하기 전에 지불 의향을 검증합니다.

- **수동 세일즈**: 창업자가 직접 첫 10명의 고객을 확보
- **가격 시그널**: 첫날부터 과금 — 무료 플랜은 학습을 지연시킴
- **채널**: 이미 문제를 표명한 사람들에게 직접 아웃리치
- **검증 게이트**: 큰 할인 없이 5명 이상이 결제 = 시그널 확인

## $1K~$10K MRR: 가격 설정과 리텐션

획득을 스케일하기 전에 유닛 이코노믹스를 최적화합니다.

- **가격 실험**: 2~3배 인상 테스트; 전환율과 리텐션 양쪽에 미치는 영향 추적
- **초기 이탈 감사**: 처음 30일 내 이탈자의 퇴출 설문 + 세션 녹화
- **온보딩 최적화**: 타임 투 밸류 단축; 활성화 마일스톤 추적
- **업그레이드 트리거**: 플랜 업그레이드를 예측하는 사용 패턴 식별

## $10K~$50K MRR: 스케일링 시스템

창업자 중심에서 시스템 중심 성장으로 전환합니다.

- **활성화 퍼널**: 가입부터 "아하 모먼트"까지의 핵심 경로를 매핑하고 최적화
- **확장 매출**: 업셀, 애드온 기능, 시트 확장을 위한 사용량 기반 트리거
- **첫 마케팅 채용**: 콘텐츠 + 유료 + 라이프사이클을 운영할 수 있는 제너럴리스트
- **채널 다각화**: 초기 채널 외에 1~2개의 획득 채널 추가

## 리텐션 레버

최고의 성장 레버는 기존 고객 유지입니다.

- **코호트 분석**: 가입 코호트별 주간 리텐션 커브 — 회귀를 조기에 발견
- **NPS 기반 개선**: 디트랙터 피드백을 제품 로드맵에 직접 라우팅
- **리인게이지먼트 플로우**: 사용량 감소를 트리거로 하는 자동 시퀀스
- **건강 스코어링**: 30일 전에 이탈 위험을 예측하는 복합 스코어
