# SaaS成長プレイブック — MRR $0から$50Kへ

収益こそ唯一のメトリクス。このプレイブックはMRRマイルストーン別に整理されています。

## $0〜$1K MRR: 最初の有料顧客

さらに開発する前に、支払い意欲を検証する。

- **マニュアルセールス**: 創業者が最初の10人の顧客を自ら獲得
- **プライシングシグナル**: 初日から課金 — 無料プランは学びを遅らせる
- **チャネル**: すでに課題を表明している人へのダイレクトアウトリーチ
- **検証ゲート**: 大幅な割引なしで5人以上が支払い＝シグナル確認

## $1K〜$10K MRR: 価格設定とリテンション

獲得をスケールする前にユニットエコノミクスを最適化。

- **価格実験**: 2〜3倍の値上げをテスト。コンバージョンとリテンション両方への影響を追跡
- **初期チャーン監査**: 最初の30日以内に離脱したユーザーの退出アンケート＋セッション録画
- **オンボーディング最適化**: タイムトゥバリューを短縮。アクティベーションマイルストーンを追跡
- **アップグレードトリガー**: プランアップグレードを予測する利用パターンを特定

## $10K〜$50K MRR: スケーリングシステム

創業者駆動からシステム駆動の成長へシフト。

- **アクティベーションファネル**: サインアップから「アハモーメント」への重要パスをマッピングして最適化
- **エクスパンション収益**: アップセル、アドオン機能、シート拡張のための使用量ベーストリガー
- **最初のマーケティング採用**: コンテンツ＋有料＋ライフサイクルを運用できるジェネラリスト
- **チャネル多角化**: 最初のチャネルに加えて1〜2つの獲得チャネルを追加

## リテンションレバー

最大の成長レバーは既存顧客の維持。

- **コホート分析**: サインアップコホート別の週次リテンションカーブ — 回帰を早期に発見
- **NPS駆動の改善**: デトラクターのフィードバックをプロダクトロードマップに直接ルーティング
- **リエンゲージメントフロー**: 使用量低下をトリガーとした自動シーケンス
- **ヘルススコアリング**: 30日先のチャーンリスクを予測するコンポジットスコア
