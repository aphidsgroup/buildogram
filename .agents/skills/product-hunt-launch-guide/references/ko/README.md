# Product Hunt 런칭 가이드 — 처음 하는 분을 위한 단계별 SOP

첫 Product Hunt 런칭에 필요한 모든 것을 실행 가능한 체크리스트로 정리했습니다. 30회 이상의 #1 Product of the Day 달성 경험에 기반합니다.

| 일 | 태스크 |
|----|--------|

| 3 | 헌터 후보 5명 선정 |

| 6-7 | PH 커뮤니티에서 활동 시작 |

| 일 | 태스크 |
|----|--------|

| 26-28 | 포스트 런칭 팔로업 |

## 헌터 전략
- 팔로워 1000명 이상이고 최근 활동이 있는 헌터를 타깃
- 피치 내용: 프로덕트 링크 + 왜 그 오디언스에게 가치 있는지
- 독점적 얼리 액세스 제공
- 스팸 금지 — 최대 3명, 개인화된 메시지

## 에셋 체크리스트
- [ ] 썸네일: 명확하고 대담하며 프로덕트가 보이는
- [ ] 태그라인: 60자 이내, 혜택 중심, 유행어 배제
- [ ] 갤러리: 핵심 기능 보여주는 이미지 5장 + 데모 영상 1개
- [ ] 첫 댓글: 500단어 이상, 스토리 기반, 로드맵 포함
- [ ] 설명: 문제 → 솔루션 → 차별화 요소
