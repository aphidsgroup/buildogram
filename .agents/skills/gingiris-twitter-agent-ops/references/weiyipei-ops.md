## Voice Guide 铁律要点（@WeiYipei 实例化）

SKILL.md 第二节的铁律模板在 @WeiYipei 账号上的落地版：

- **铁律 A**：开头必须是「我」+ 具体经历/数字/瞬间（6 条爆款全部第一人称开头）
- **铁律 B**：禁止三段论（论点→论据→号召）结构
- **铁律 C**：不是每条都要有"总结"
- **铁律 D**：粗体 = 信念表达，不是重点标注
- **死亡开头黑名单**：引用他人话语 / 大命题式（"AI时代一个反直觉的事"）/ 无主语大词 / 哲理金句直接开头
- **内容权重**：长文（个人经历+数据+洞察）70% / 工具资源帖 20% / 生活碎片 10%
- **发布窗口**：早 8 点（实测爆款命中率 5%→15%），17/23 点为二级窗口
- **数字来源**：只用 SOURCE-INDEX 里标 ✅ 的素材点（如"开源第一周 6000 Star"= ep01 第77行）
