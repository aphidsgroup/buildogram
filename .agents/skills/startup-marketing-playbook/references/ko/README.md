# 예산 0원 스타트업 마케팅 — 오가닉 채널만으로

돈이 없다고? 문제없다. 여기 있는 모든 전술은 0원 — 필요한 건 시간과 창의력뿐.

## 맨땅에서 시작하는 SEO

- 백링크도 도메인 권위도 없는 상태에서 검색 순위를 얻는 방법
- 예산 0원 SEO는 돈이 아니라 시간의 투자다. '준비되면'이 아니라 지금 시작할 것

## 커뮤니티 마케팅 (스팸이 아니라)

- 기존 커뮤니티 안에서 홍보가 아닌 진짜 가치를 제공한다
- 지켜야 할 선: 먼저 기여하고, 제품 언급은 그것이 누군가의 문제에 실제 답이 되는 순간에만
- 가치 우선 참여만이 커뮤니티에서 환영받는 길이다. 스팸은 퇴출과 채널 소각으로 끝난다

## 소셜 미디어 플레이북

- 플랫폼별로 최적화한 콘텐츠로 오가닉 가입을 만들어낸다
- 같은 글을 모든 곳에 복붙하지 말 것. 플랫폼마다 보상받는 콘텐츠의 형태가 다르다

## 입소문 설계

- 제품 자체를 자연스럽게 공유하고 싶은 것으로 만들어, 사용자가 마케팅을 대신하게 한다
- 입소문은 나중에 덧붙이는 기능이 아니라 제품 경험 안에 설계해 넣는 것

## 게릴라 전술

- 0원으로도 체급 이상의 성과를 내는 창의적이고 악착같은 접근법
- 제약이 창의력을 낳는다. 예산이 없기에 포지셔닝이 더 날카로워진다

## B2B vs 2C 보정

이 플레이북의 기본값은 개발자 도구/오픈소스/B2B SaaS. 제품이 2C(컨슈머·교육·앱·게임)라면:

- 핵심 지표를 D1/D7/D30 리텐션, 활성화율, 바이럴 계수 K로 교체 (MRR/CAC/LTV가 아니라)
- 콜드스타트 채널을 버티컬 커뮤니티, 숏폼 영상, 버티컬 KOL로 교체 (Product Hunt / Hacker News / LinkedIn이 아니라)
- 국가별 채널 데이터가 포함된 완전한 2C 적응 가이드는 `gingiris-seo-geo/references/2c-adaptation.md` 참조
