# 予算ゼロのスタートアップ・マーケティング — オーガニックチャネルのみ

お金がない？問題ない。ここにある戦術はすべて 0 円——必要なのは時間と創造力だけ。

## ゼロから始める SEO

- 被リンクもドメインオーソリティもない状態から検索順位を取る方法
- 予算ゼロの SEO はお金ではなく時間の投資。「準備ができてから」ではなく今日から始める

## コミュニティマーケティング（スパムではなく）

- 既存コミュニティの中で、宣伝ではなく本物の価値を提供する
- 守るべき一線：まず貢献する。プロダクトに言及するのは、それが誰かの問題への本当の答えになる場面だけ
- 価値提供ファーストだからこそ居場所が保てる。スパムは追放とチャネルの焼失を招くだけ

## ソーシャルメディア・プレイブック

- プラットフォームごとに最適化したコンテンツで、オーガニックに登録を生む
- 同じ投稿の使い回しはしない。プラットフォームごとに評価されるコンテンツの形が違う

## 口コミの設計

- プロダクト自体を自然にシェアしたくなるものにして、ユーザーにマーケティングをしてもらう
- 口コミは後付けの機能ではなく、プロダクト体験の中に設計するもの

## ゲリラ戦術

- 予算 0 円でも見た目以上の成果を出す、創造的で泥臭いアプローチ
- 制約は創造性を生む。予算がないからこそポジショニングが研ぎ澄まされる

## B2B と 2C の使い分け

本プレイブックのデフォルトは開発者ツール／オープンソース／B2B SaaS。プロダクトが 2C（コンシューマー・教育・アプリ・ゲーム）の場合：

- コア指標を D1/D7/D30 リテンション・アクティベーション率・バイラル係数 K に置き換える（MRR/CAC/LTV ではなく）
- コールドスタートのチャネルを垂直コミュニティ・ショート動画・専門 KOL に置き換える（Product Hunt / Hacker News / LinkedIn ではなく）
- 国別チャネルデータを含む完全な 2C 適応ガイドは `gingiris-seo-geo/references/2c-adaptation.md` を参照
