## Community Welcome Post

```
👋 Welcome to [Community Name]!

[One-line brand intro + core community value]

🔗 Official resources:
- Brand website: [link]
- Help center / Docs: [link]
- Templates / Resources: [link]

📋 Community rules:
1. 💣 No unauthorized advertisements
2. 🚫 No irrelevant content
3. 🤝 Respect others — no personal attacks or hate speech
4. ✅ Verify before sharing — ensure content is real and valid

Introduce yourself below 👇 Where are you from and how did you find us?
```

## Ambassador Application Form (Typeform / Google Form)

Fields to include:

1. Name (required)
2. Email (required)
3. Location — city + country (required)
4. Social media accounts — Twitter/X, LinkedIn, YouTube, etc.
5. **"Why do you love [brand]?"** (required text box — key filter for genuine users)
6. Where do you usually share content? (multi-select: Twitter/X, LinkedIn, YouTube, TikTok, Newsletter, Blog, Reddit, Other)
7. Interested contribution types (multi-select): Content creation / Community support / User feedback / Localization/translation / Bug reports / Event organization
8. **"How do you plan to contribute to the community?"** (required text box — evaluates commitment and creativity)
9. Weekly time available (single select): <1h / 1–3h / 3–5h / 5+ hours
10. Event organization experience? (Yes/No + describe if yes)
11. Content creation experience? (Yes/No + provide link if yes)

Reference forms: https://6dxre9ihosp.typeform.com/to/CLw1pUAp

## Community Rules Template (Full)

```
📋 [Community Name] Community Guidelines

Welcome! To keep this a great place for everyone:

✅ DO:
- Share your genuine Lark/[brand] experiences and questions
- Help other members when you can
- Share useful resources, templates, and tutorials
- Provide constructive feedback
- Disclose if you're an official ambassador or partner when relevant

❌ DON'T:
- Post ads or promotional content without permission
- Share misleading or unverified information
- Make personal attacks or use hate speech
- Leak confidential product information
- Post competitor promotion in this community

⚠️ Violations:
- Minor (first offense): Private message + guidance
- Repeated minor: Written warning, temporary benefit suspension
- Severe (false info, harassment): Immediate removal from community

Questions? DM the admin team.
```
