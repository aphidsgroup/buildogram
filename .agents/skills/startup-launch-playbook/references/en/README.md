# Early-Stage Startup Launch — Pre-Seed to First 1000 Users

A tactical guide for founders who've built something and need their first real users. No theoretical fluff — just the channels and tactics that actually work at each stage.

## MVP Validation
- Run 5 problem interviews before writing a single line of code
- Build a "concierge MVP" — manually deliver the value before automating
- Set a kill criteria: if X doesn't happen by Y date, pivot or stop
- Validate willingness to pay, not just interest

## First 10 → 100 → 1000 Users

### First 10 Users (Week 1-2)
| Channel | Tactic | Expected Result |
|---------|--------|-----------------|
| Personal network | DM friends who fit ICP | 3-5 users |
| Communities | Post in relevant Discord/Slack | 3-5 users |
| Twitter/X | Build-in-public thread | 2-3 users |

### First 100 Users (Week 3-6)
| Channel | Tactic | Expected Result |
|---------|--------|-----------------|
| Product Hunt | Small launch for feedback | 30-50 users |
| Reddit | Value-first posts in niche subs | 20-30 users |
| Cold outreach | 50 personalized DMs/emails | 10-20 users |

### First 1000 Users (Month 2-3)
| Channel | Tactic | Expected Result |
|---------|--------|-----------------|
| Content marketing | 2 high-value posts/week | 200-300 users |
| KOL partnerships | 3-5 micro-influencer deals | 200-400 users |
| Referral loop | Built-in sharing mechanism | 200-300 users |

## Founder-Led Sales Templates
- Cold DM template (Twitter): "Hey [name], I noticed you [specific thing]. I built [tool] that [benefit]. Would you try it free for a week?"
- Cold email template: Problem → Solution → Social proof → Soft CTA
- Follow-up sequence: Day 1, Day 3, Day 7 (max 3 touches)

## Traction Metrics by Stage
| Stage | Key Metric | Target |
|-------|-----------|--------|
| Pre-seed | Weekly active users | 50+ WAU |
| Seed | MoM growth rate | 15-20% |
| Series A | Revenue or engagement | $10k MRR or 10k DAU |

## Bootstrapping Tactics
- Trade equity/service for marketing help
- Use free tiers of tools (Vercel, Supabase, Resend)
- Content marketing > paid ads at early stage
- Partner with complementary products for co-marketing
