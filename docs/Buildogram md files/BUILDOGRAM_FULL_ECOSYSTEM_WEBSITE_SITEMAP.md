# BUILDOGRAM FULL ECOSYSTEM WEBSITE SITEMAP

# Purpose

This document defines the complete website sitemap for Buildogram's full ecosystem launch.

Buildogram should not look like a normal construction company website.

The website must show Buildogram as:

- Construction platform
- Material marketplace
- Partner ecosystem
- Property Passport platform
- 360 rental and resale property platform
- Maintenance platform
- AI construction/property tools
- Content and education engine

Core sitemap principle:

**Every major business vertical should have a clear public page, lead form and conversion path from Day 1.**

---

# Website Positioning

## Main Website Identity

Buildogram should present itself as:

**India's Property Transparency Platform**

Not:

- Contractor website
- Builder website
- Real estate website
- Material shop website
- Agency website

Buildogram combines all of them into one property lifecycle system.
